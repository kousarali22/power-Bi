Power BI Sales Analysis Dashboard
Project Overview

This project focuses on analyzing sales data and presenting insights through interactive Power BI dashboards. The goal is to transform raw data into clear, visual reports to support data-driven decision-making.

Tools Used

Power BI

Power Query

Microsoft Excel / CSV files

Key Features

Interactive dashboards using charts, tables, slicers, and filters

Data cleaning and transformation using Power Query (ETL)

Integration of data from Excel/CSV sources

Trend analysis and summary-level insights

Clean formatting for improved data readability

Data Processing

Removed duplicates and handled missing values

Renamed columns and standardized data formats

Applied basic aggregations and sorting

Outcome

The dashboard provides an easy-to-understand visual summary of sales trends and performance, enabling quick insights without complex calculations.
